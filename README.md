# 🚗 AutoStore

Sistema web para uma concessionária de veículos com catálogo dinâmico e assistente virtual inteligente utilizando Gemini AI, Embeddings e RAG.

## 📖 Sobre o projeto

A AutoStore foi desenvolvida para facilitar a busca por veículos através de um catálogo online integrado a um chatbot inteligente.

O sistema permite pesquisar veículos, visualizar informações detalhadas, registrar interesse em um carro e conversar com uma IA capaz de responder perguntas utilizando apenas os dados do catálogo.

## ✨ Funcionalidades

- Catálogo de veículos
- Pesquisa por modelo e categoria
- Página de detalhes
- Galeria de imagens
- Cadastro de leads
- Chatbot com Gemini
- Embeddings
- Busca semântica (RAG)
- Histórico de conversa

## 🚀 Como executar

1. Clone o repositório

git clone ...

2. Coloque a pasta dentro do XAMPP

3. Inicie Apache e MySQL

4. Importe o banco de dados

5. Configure a chave da API Gemini em config.php

6. Acesse:

http://localhost/AutoStore

## 🤖 AutoIA

A AutoIA utiliza a API Gemini juntamente com Embeddings e RAG para responder perguntas sobre os veículos cadastrados.

O chatbot consulta o catálogo da concessionária e responde apenas com informações presentes na base de dados, evitando recomendações de veículos inexistentes.
